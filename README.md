# Number Guessing Game 🎯

A simple number guessing game built with HTML, CSS & JavaScript.

## Features
- Guess a number between 0 and 1000
- View guessing history
- Play again option

## Technologies
- HTML5
- CSS3
- JavaScript

## How to play
1. Enter a number between 0-1000
2. Click "ثبت عدد" (Submit)
3. Read the hint to guess higher or lower
4. You have 10 chances
5. Win by guessing the correct number!

## How to run

Open https://luppin-1.github.io/Numbers-Game/ in your browser.
